Challenge Source Tracking

Track origins from Work Manager, Topgear, Github, etc.

Available Endpoints

GET /api/v6/challenges

Get all challenges with filtering and pagination

Supports filtering by challengeSource, status, type, track
POST /api/v6/challenges

Create a new challenge

Includes challengeSource field for origin tracking
GET /api/v6/challenges/{id}

Get challenge by ID

PUT /api/v6/challenges/{id}

Update challenge (full update)

challengeSource field can be updated
PATCH /api/v6/challenges/{id}

Partially update challenge

DELETE /api/v6/challenges/{id}

Delete challenge

Challenge Source Feature

The new challengeSource field allows tracking the origin of challenges.

  • Optional string field
  • Can be null or any text value
  • Supports filtering in search queries
  • Included in create/update operations
Example Values:
  • • Work Manager
  • • Topgear
  • • Github
  • • Manual Entry
  • • API Integration
